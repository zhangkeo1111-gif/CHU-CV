# CV-for-CHUer
专为长安大学学生与 CHUer 打造的 LaTeX/XeLaTeX 中文简历模板，可用于保研、求职、实习申请、竞赛报名及各类材料投递。模板设计主要参考了 NPU-CV、NCU-CV 以及一些经典中文简历模板，在此基础上融入长安大学风格的页眉页脚、蓝色学术化版式与模块化排版结构，支持个人信息、教育背景、荣誉奖项、竞赛经历、实习经历、项目经历、技能特长等常用简历内容。  仓库中提供了乌萨奇主题的二创示例，用更轻松的方式展示模板效果：从文字替换、头像修改，到 Logo、背景图、联系方式和模块内容调整，乌萨奇手把手带大家做出一份精美、清晰、适合投递的中文简历 PDF。  本模板仍在持续完善中，如在使用过程中遇到问题、排版错误或其他 bug，欢迎联系反馈，也欢迎提交 issue 或 PR 一起改进。
